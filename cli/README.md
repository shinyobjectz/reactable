# reactable-cli

Agent CLI for [Reactable](https://reactable.app) — decks, takes, stage, YouTube.

Requires [Bun](https://bun.sh) ≥ 1.0.

```bash
npm install -g reactable-cli
reactable doctor
reactable youtube connect
reactable stage open --deck demo
```

Full docs: [github.com/shinyobjectz/reactable](https://github.com/shinyobjectz/reactable)
