# Decks — agent verbs

Load when authoring or restructuring `deck.work`.

## Commands

```bash
reactable decks list
reactable decks get <slug> --json
reactable decks new "Title"
reactable decks validate <slug>
reactable decks slide add|move <slug> …
reactable decks script add|run <slug> …
reactable plan <slug>
```
