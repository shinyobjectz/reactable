# Podcast Cover

## Use when

Use for square cover art that must remain legible at small sizes and support title branding.

## Signature traits

- square balance
- centered brand clarity
- title-safe layout
- graphic simplicity with personality

## Prompt levers

- keep title area quiet and intentional
- use strong silhouette or icon shape
- avoid background noise near wordmarks

## Examples

### Example 1 — Host portrait cover

Centered host with simplified color field backdrop and bold title band.

### Example 2 — Abstract science show cover

Clean icon-led composition with orbital motif and large readable title.

## Avoid

- busy edge clutter
- tiny decorative copy that no listener will ever read
