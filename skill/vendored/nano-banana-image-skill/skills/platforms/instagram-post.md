# Instagram Post

## Use when

Use for square or portrait social posts that need immediate visual readability and strong center-of-interest design.

## Signature traits

- fast focal read
- clear subject silhouette at small size
- crop resilience
- clean edge clutter control

## Prompt levers

- prefer 1:1 or 4:5
- keep the main subject large enough for mobile viewing
- reserve simple space if text overlays are needed

## Examples

### Example 1 — Portrait post

4:5 crop with face occupying strong central area and background detail simplified.

### Example 2 — Product post

Centered hero object with clean negative space for caption-safe visual breathing room.

## Avoid

- tiny focal subjects
- dense fine detail that dies at thumbnail scale
