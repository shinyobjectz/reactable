# Ray Tracing

## Use when

Use when reflective accuracy, hard-surface realism, and direct lighting interactions matter visibly.

## Signature traits

- accurate reflection behavior
- clean hard-surface light interaction
- sharper realism in reflective materials

## Prompt levers

- best on glass, metal, lacquer, and polished environments
- use only when the frame benefits from visible reflection logic

## Examples

### Example 1 — Product bottle

Glass and metal surfaces with believable reflection paths and controlled highlight structure.

### Example 2 — Sci-fi visor

Reflective helmet visor showing coherent environment reflections.

## Avoid

- adding it to matte scenes where it changes almost nothing
