# Path Tracing

## Use when

Use when richer light transport, interior bounce, or premium CG realism matters.

## Signature traits

- soft indirect bounce realism
- physically rich global illumination
- premium studio or interior light behavior

## Prompt levers

- especially useful for interiors and product scenes
- keep it subordinate to actual composition and material choices

## Examples

### Example 1 — Studio tabletop

Elegant product scene with nuanced bounce off white cards and subtle floor return.

### Example 2 — Interior render

Warm lamp-lit room with believable secondary light spill on walls and furniture.

## Avoid

- name-dropping it in flat outdoor scenes that do not need the cue
