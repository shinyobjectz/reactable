# Ukiyo-e

## Use when

Use when you need woodblock-inspired line rhythm, flattened depth, pattern discipline, and elegant cropping.

## Signature traits

- flattened but deliberate spatial layers
- contour-led shape rhythm
- decorative pattern blocks
- controlled color separation

## Prompt levers

- protect broad negative shapes and silhouette clarity
- let line and pattern carry texture instead of photographic microdetail
- compose with asymmetry and graceful cropping

## Examples

### Example 1 — Festival bridge print

Lantern procession crossing a bridge with elegant wave patterns, flat water planes, and decorative cloud or mist forms.

### Example 2 — Portrait panel

Half-length figure with kimono pattern emphasis, controlled background plane, and strong contour cadence.

## Avoid

- photoreal depth blur
- modern glossy lighting language that fights the print logic
