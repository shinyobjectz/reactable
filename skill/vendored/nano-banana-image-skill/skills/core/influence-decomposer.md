
# Influence Decomposer

## Purpose

Translate influence requests into formal traits the model can follow reliably.

Do not rely on naked name-dropping.
Turn influences into:
- palette,
- line behavior,
- shape language,
- texture character,
- staging logic,
- contrast structure,
- movement or stillness.

## Example 1 — "like a prestige sci-fi film"

Decompose into:
- restrained palette with selective saturated accents,
- practical light sources embedded in production design,
- engineered surfaces with wear at touch points,
- strong silhouette readability,
- camera restraint rather than chaotic distortion.

## Example 2 — "like an Edo print"

Decompose into:
- flattened depth,
- rhythmic contour lines,
- bold pattern blocks,
- decorative but disciplined cropping,
- controlled negative space,
- woodblock-like color separation.

## Example 3 — "like a luxury tech ad"

Decompose into:
- immaculate surface control,
- sparse composition,
- measured reflections,
- precision typography zones,
- cool-neutral palette with one accent color.

## Use this when

- the user references a genre, mood board, or broad influence,
- the named inspiration is too vague,
- direct mimicry is undesirable or risky,
- you want reusable language across different models.

## Quick move

Replace:
> in the style of X

with:
> using the following traits from X: [trait list]

That produces stronger prompts and less superstition.
