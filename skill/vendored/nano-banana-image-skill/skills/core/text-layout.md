
# Text Layout

## Purpose

Treat text-in-image as a layout system, not a magic word spell.

## Always specify

- exact copy,
- hierarchy,
- placement,
- safe margins,
- background quietness,
- whether text must be fully readable or only suggestive.

## Hierarchy roles

- headline
- subhead
- body
- CTA
- legal / sponsor block
- label text

## Example 1 — festival poster

Headline: `Lanterns of Spring`  
Placement: upper third  
Safe margins: 8–10% on all sides  
Background: simplified sky and lantern glow behind title field  
Secondary copy zone: lower third

## Example 2 — podcast cover

Headline: show title centered  
Secondary: host names smaller below  
Square crop, strong central subject, quiet edge zones, no busy details behind wordmarks.

## Example 3 — infographic

Use modular panels, labels near the data they describe, larger title at top, restrained color coding, clean icon spacing, and lots of breathing room.

## Text rules

- Reserve space on purpose.
- Do not place key text over high-frequency detail.
- If readability matters, prefer Pro.
- For labels or product packaging, specify exact placement and scale relationship.
- Keep each text phrase short when possible (roughly <=25 characters per phrase).
- Prefer 1–3 phrases total for clean compositions; more text should be treated as layout-heavy and iterated in passes.

## Failure modes

- "include the text somewhere"
- no copy hierarchy
- beautiful background directly behind tiny type
- too many words for the available layout

## Repair move

If text is struggling, simplify the scene behind the type and increase hierarchy contrast.
