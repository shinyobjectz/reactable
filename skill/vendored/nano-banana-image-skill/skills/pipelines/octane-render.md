# Octane Render

## Use when

Use when the image should feel like premium offline CG with controlled speculars, rich materials, and advertisement-grade polish.

## Signature traits

- clean high-end CG finish
- precise reflection behavior
- luxury material rendering
- studio-grade surface readability

## Prompt levers

- great for products, hero props, and controlled environments
- pair with path-tracing when the light transport matters visibly
- keep lighting measured rather than theatrical by default

## Examples

### Example 1 — Product still

Luxury fragrance bottle with frosted glass and restrained caustic-like shimmer.

### Example 2 — Prop render

Hero helmet on dark pedestal with disciplined side reflections and subtle haze.

## Avoid

- combining with every other pipeline keyword
- casual phone-camera artifacts unless a hybrid aesthetic is intentional
