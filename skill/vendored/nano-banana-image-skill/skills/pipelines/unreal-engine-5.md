# Unreal Engine 5

## Use when

Use when you want high-end real-time cinematic CG cues: game-engine staging, environmental scale, and glossy modern digital lighting.

## Signature traits

- engineered cinematic environments
- clean real-time CG polish
- strong emissive and atmospheric integration
- game-adjacent camera logic

## Prompt levers

- keep geometry and set dressing legible at frame scale
- pair with volumetric-lighting when visible
- avoid overstacking pipeline labels

## Examples

### Example 1 — Sci-fi corridor

Hero corridor with emissive signage, metallic wear at touch points, and dramatic but readable atmosphere.

### Example 2 — Fantasy ruin

Large ruin vista with layered fog shafts and game-key-art camera placement.

## Avoid

- using UE5 as meaningless prestige garnish
- forcing fully offline-render language when real-time CG is the point
