# Manga Base

## Use when

Use for monochrome or graphic storytelling with line rhythm, value control, and panel-aware composition.

## Signature traits

- crisp inked contours
- strong black-white value grouping
- screentone or halftone logic when useful
- graphic focus on gesture and emotion

## Prompt levers

- decide whether the output is poster-like, cover-like, or panel-like
- choose line density and screentone restraint
- protect negative space for text or title blocks

## Examples

### Example 1 — Cover image

Three-quarter portrait with graphic shadow blocks, strong type-safe masthead region, elegant line tapering.

### Example 2 — Historical panel

Bridge crowd scene rendered with flattened depth and ink rhythm that preserves readable character grouping.

## Avoid

- photographic bokeh jargon unless the manga finish truly needs a camera-derived look
- soft gray mush with no value hierarchy
