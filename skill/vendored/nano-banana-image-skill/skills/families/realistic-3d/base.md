# Realistic 3D Base

## Use when

Use for high-end CG that should feel rendered rather than photographed, while still reading as credible.

## Signature traits

- precise material control
- clean specular behavior
- intentional render polish
- geometry-first realism

## Prompt levers

- choose the dominant pipeline cue only when visible
- protect model geometry and seam logic
- combine with path/ray/volumetric features only when they improve the shot

## Examples

### Example 1 — Product hero

Lux bottle render with measured reflections, frosted glass, and clean studio sweep.

### Example 2 — Game key art

Mech corridor establishing frame with layered emissive signage and disciplined atmospheric depth.

## Avoid

- photographic accident language unless the CG should mimic capture artifacts
- stacking every pipeline buzzword for no reason
