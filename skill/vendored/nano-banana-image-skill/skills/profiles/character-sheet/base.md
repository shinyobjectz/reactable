# Character Sheet Base

## Use when

Use for turnaround sheets, expression sheets, costume callouts, and continuity-driven character documentation.

## Signature traits

- consistency across views
- clean layout with labeled zones
- silhouette and costume-lock discipline

## Prompt levers

- state required views explicitly
- protect emblem placement and seam logic
- leave quiet space for notes or callouts

## Examples

### Example 1 — Turnaround sheet

Front, three-quarter, side, and back views with the same suit seams, boots, gloves, and helmet proportions.

### Example 2 — Expression set

Neutral, determined, amused, and alarmed facial studies in a stable head angle grid.

## Avoid

- dramatic lighting that hides reference details
- layout clutter that undermines the sheet function
