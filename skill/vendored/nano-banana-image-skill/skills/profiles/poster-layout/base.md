# Poster Layout Base

## Use when

Use when the image must function as poster art with title hierarchy, empty zones, and long-distance readability.

## Signature traits

- title-safe composition
- clear hierarchy from far away
- focal subject balanced against text fields

## Prompt levers

- state title placement early
- protect upper and lower text bands intentionally
- simplify background detail behind critical type

## Examples

### Example 1 — Festival poster

Upper title field, midframe procession, lower venue block, decorative edges restrained.

### Example 2 — Film key poster

Large face or hero silhouette with supporting environment compressed into readable shapes.

## Avoid

- painting everything at equal intensity
- discovering at the end that there is nowhere for text to go
