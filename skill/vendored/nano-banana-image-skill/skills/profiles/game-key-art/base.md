# Game Key Art Base

## Use when

Use for hero splash imagery, store art, or promotional visuals with character drama and environmental scale.

## Signature traits

- hero-forward composition
- high silhouette contrast
- environment supporting scale and stakes
- controlled spectacle

## Prompt levers

- state hero, threat, and environment relationship clearly
- let the composition sell gameplay fantasy without becoming UI clutter
- pair well with realistic-3d or stylized-3d

## Examples

### Example 1 — Mech confrontation

Hero mech foregrounded against scale-rich industrial canyon and distant threat silhouettes.

### Example 2 — Fantasy squad art

Frontline heroes, readable weapon silhouettes, atmospheric battlefield depth, banner-safe title zone.

## Avoid

- tiny hero characters swallowed by effects
- chaotic value structure with no focal path
