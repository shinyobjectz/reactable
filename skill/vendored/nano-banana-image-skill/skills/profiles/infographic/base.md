# Infographic Base

## Use when

Use for educational, technical, or explanatory visuals that combine image-making with structured labeling.

## Signature traits

- high legibility
- clear visual hierarchy
- panel or module logic
- trustworthy spacing and labeling

## Prompt levers

- simplify decorative background details
- give labels room to breathe
- use color coding in moderation

## Examples

### Example 1 — Planet comparison

Modular panels comparing exoplanet classes with clean labels, icons, and scale bars.

### Example 2 — Process explainer

Step-by-step diagram with one dominant path and support annotations.

## Avoid

- ornamental overload
- poster drama that harms comprehension
