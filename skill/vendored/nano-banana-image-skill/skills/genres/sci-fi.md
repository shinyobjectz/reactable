# Sci-Fi

## Use when

Use when technology, speculative architecture, interfaces, engineered surfaces, or futuristic world systems are central.

## Signature traits

- designed technology grammar
- engineered materials and interfaces
- speculative but coherent world-building
- lighting motivated by practicals, displays, or infrastructure

## Prompt levers

- decide whether the sci-fi is clean, industrial, utopian, decayed, or neon-dense
- name what technology does, not only how it glows
- keep interface density subordinate to composition

## Examples

### Example 1 — Neon alley

Service corridor below stacked apartments, maintenance drones overhead, bilingual LED signs, wet shutters, and coded access doors.

### Example 2 — Orbital classroom infographic

Cutaway station classroom with modular displays, clear icon panels, and color-coded planetary data zones.

## Avoid

- glowing nonsense with no technology logic
- UI clutter that buries the actual subject
