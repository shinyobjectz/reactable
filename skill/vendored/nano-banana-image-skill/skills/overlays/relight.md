# Relight

## Use when

Use when geometry and composition remain stable but the light mood changes.

## Signature traits

- unchanged structural image
- new light logic
- material-response continuity

## Prompt levers

- lock crop, pose, and geometry first
- specify new key/fill/rim structure
- state atmosphere and color temperature shift

## Examples

### Example 1 — Day to storm twilight

Keep scene exact, shift to cool sky fill with cyan rim and thin rain haze.

### Example 2 — Flat studio to premium hero

Keep product position exact, upgrade to controlled top softbox and elegant edge separation.

## Avoid

- rewriting background layout
- changing materials so much the object looks replaced
