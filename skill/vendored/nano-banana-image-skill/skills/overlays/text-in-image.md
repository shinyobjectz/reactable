# Text in Image

## Use when

Use when the image itself must contain readable words, labels, titles, or supporting copy.

## Signature traits

- layout-conscious composition
- reserved negative space
- hierarchy-driven text planning

## Prompt levers

- state exact copy and hierarchy
- reserve quiet fields behind key text
- prefer Pro when readability is non-negotiable

## Examples

### Example 1 — Poster title

Large headline in upper third with simplified sky or gradient field behind it.

### Example 2 — Product label

Exact front-label text preserved with realistic placement and scale.

## Avoid

- asking for long copy with no layout plan
- placing type over noisy texture
