## In-Image Translation (New in 2026)

### Full image localization:
```
Translate all visible text in this image from [SOURCE LANGUAGE] to [TARGET LANGUAGE].
Preserve exactly: all visual design, colors, layout, imagery, proportions.
For each text element: match the original font weight (bold/regular), size, and color.
Do not rearrange any elements - only replace the text content.
Final image must be indistinguishable in design from the original, only language changes.
```

### Selective text translation:
```
In this image, translate ONLY the [SPECIFIC TEXT ELEMENT, e.g. "headline at top"]
from [SOURCE LANGUAGE] to [TARGET LANGUAGE].
Keep all other text in the original language.
Keep everything else unchanged: layout, colors, all other text, imagery.
Match font style as closely as possible.
```
