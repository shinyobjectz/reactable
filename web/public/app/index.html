YouTube

Uses the same OAuth as studio: ~/.shinyobjectz/youtube_client.json and youtube_token.json. Desktop / CLI connect via loopback — not cloud redirect yet.

reactable youtube connect
reactable youtube search "reaction clip"
reactable youtube proxy --video aqz-KE-bpKQ

Cloud dashboard YouTube connect (web redirect) ships when a Web OAuth client is added to your Google Cloud project.

CLI

Authenticate the desktop app and agent from your terminal:

reactable auth login
reactable youtube search "podcast clip"
reactable youtube proxy --video aqz-KE-bpKQ
